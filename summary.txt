STEP 3 :  Cognitive Decomposition**
- **Inbound Orchestrator**: Valida la coerenza del `brief.json` e l'integrità degli asset grafici.
• **Creative AI Engine**: Trasforma i requisiti in varianti UI e contenuti testuali multilingua.
• **SEO Semantic Agent**: Inietta metadati localizzati e ottimizza i contenuti per le keyword regionali.
• **GEO Deployment Hub**: Gestisce il rilascio globale, garantisce la conformità GDPR e attiva la telemetria.

STEP 4. Architectural Layout : 
• InboundOrchestrator: Next.js Server Actions
• CreativeAIEngine: Vercel Edge Functions
• SEOSemanticAgent: Next.js API Routes
• GEODepHub: Vercel Edge Middleware

STEP 5. Integration Patterns :
• Sync Request-Response: InboundOrchestrator chiama CreativeAIEngine via fetch Edge per feedback immediato sulla validazione degli asset.
• Async Task Distribution: CreativeAIEngine attiva il SEOSemanticAgent tramite Supabase Webhooks per il processamento SEO parallelo e asincrono.
• State Persistence Pattern: Ogni modulo aggiorna lo stato su Supabase (DB-centric coordination) per garantire resilienza e tracciabilità del workflow.
• Middleware Routing: GEODepHub intercetta le richieste via Edge Middleware per applicare logiche GEO/GDPR prima del rendering.
